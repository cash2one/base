教程¶

你想要用 Python 和 Flask 开发一个应用吗？这里你有机会通过实例学习。在这个教程中我们将会创建一个简单的微博客的应用。 它仅支持一个用户可以创建纯文本条目，并且没有推送或者评论，但是它仍然有你开始时需要的一切。 我将会使用 Flask 以及来自 Python 框架的 SQLite 作为数据库，所以你不需要其它东西了。

如果你想提前获得完整源码或是用于比较，请查阅 example source 。
